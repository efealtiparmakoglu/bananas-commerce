## рҹҢҹ Features

### рҹҺЁ Frontend (Customer)
- вҡЎ **Next.js 14** with App Router
- рҹҺЁ **Tailwind CSS** + Headless UI
- рҹ“ұ **Responsive Design** (Mobile-first)
- рҹ”Қ **Advanced Search** with Elasticsearch
- рҹӣ’ **Real-time Cart** with Redis
- рҹ’і **Multi-payment** (Stripe, PayPal, Crypto)
- рҹҢҚ **Multi-language** (i18n)
- рҹҢҷ **Dark/Light Mode**

### рҹӣ пёҸ Backend (Microservices)
- рҹ”җ **Auth Service** - JWT + OAuth2 + 2FA
- рҹ“Ұ **Product Service** - Catalog, inventory, categories
- рҹӣ’ **Order Service** - Cart, checkout, orders
- рҹ’° **Payment Service** - Stripe, PayPal, webhooks
- рҹҡҡ **Shipping Service** - Multi-carrier integration
- рҹ“§ **Notification Service** - Email, SMS, push
- рҹӨ– **AI Service** - Recommendations, chatbot
- рҹ“Ҡ **Analytics Service** - Reports, dashboards

### рҹҸ—пёҸ Infrastructure
- рҹҗі **Docker** + Docker Compose
- вҳёпёҸ **Kubernetes** with Helm charts
- рҹ“Ҡ **Prometheus** + Grafana monitoring
- рҹ”Қ **Elasticsearch** + Kibana
- рҹ”„ **CI/CD** with GitHub Actions
- рҹ§Ә **Testing** - Unit, Integration, E2E

## рҹҡҖ Quick Start

### Prerequisites
- Node.js 18+
- Docker 20.10+
- Kubernetes 1.24+ (optional)

### Development

```bash
# Clone repository
git clone https://github.com/efealtiparmakoglu/bananas-commerce.git
cd bananas-commerce

# Start with Docker Compose
make dev-up

# Or start services individually
# Frontend
cd apps/web && npm install && npm run dev

# Backend services
cd services/auth && npm install && npm run dev
cd services/product && npm install && npm run dev
# ... (other services)
```

### Access Points

| Service | URL |
|---------|-----|
| Web App | http://localhost:3000 |
| Admin Panel | http://localhost:3001 |
| API Gateway | http://localhost:8000 |
| GraphQL Playground | http://localhost:8000/graphql |
| Prometheus | http://localhost:9090 |
| Grafana | http://localhost:3002 |
